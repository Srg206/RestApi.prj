Для начало выполните инструкцию из Guide_to_run_bd

Необходимо установить все библиотеки и модули из requirments

WINDOWS:
1)установите библиотеки с помощью vcpkg
2)выполните команду vcpkg integrate install, вы получите ключ с которым нужно собирать проект.
3)склонируйте репозиотрий
4)Создайте в нем папку build 
5) из папки build выполните команду cmake ключ(Например: "-DCMAKE_TOOLCHAIN_FILE=C:/src/vcpkg/scripts/buildsystems/vcpkg.cmake") ..
6)запустите .sln файл и запустите проект
7)запустите исполняемый файл main.exe а потом test.exe в результате в папке test должен появиться файл api_logs.txt,
показывающий status коды работы api


LINUX:
1)склонируйте репозиторий
2)создайте папку build
3)из папки build выполните команду cmake ..   а затем make
4)запустите исполняемый файл main   а затем tests  в результате в папке test должен появиться файл api_logs.txt,
показывающий status коды работы api
